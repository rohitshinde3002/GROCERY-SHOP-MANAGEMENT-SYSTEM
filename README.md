# Grocery Shop Management System

## Project Description
Grocery Shop Management System is a software project used to manage
daily activities of a grocery shop like products, billing, customers,
and stock management.

## Features
- Add, update, and delete products
- Manage stock quantity
- Generate bills
- Customer management
- Simple and user-friendly interface

## Technologies Used
- Programming Language: (Php)
- Database: (SQL)

## How to Run Project
1. Download or clone the repository
2. Open the project in your system
3. Run the main file
